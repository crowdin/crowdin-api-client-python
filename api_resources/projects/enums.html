class EscapeSpecialCharacters (value, names=None, *, module=None, qualname=None, type=None, start=1)

Defines whether any special characters (=, :, ! and #) should be escaped by backslash in exported translations. You can add escape_special_characters per-file option. *

Acceptable values are: 0 - Do not escape special characters 1 - Escape special characters by a backslash
